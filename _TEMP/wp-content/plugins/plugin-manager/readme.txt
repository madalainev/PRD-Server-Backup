== Description ==

Plugin Manager lets you to view, download and install plugins from wordpress.org from an AJAX'ed interface, instead of manually downloading, extracting and uploading each plugin.

This plugin is tested to work in Firefox 3, IE6, IE7 on a linux hosting environment.

(You may require suphp (http://www.suphp.org) to be installed so that php can download and extract plugin files)

Report All Bugs at http://www.drunkadmin.com/forums/

Plugin Database last updated 18th July 2009

== Installation ==

1. Upload the files to the '/wp-content/plugins/plugin-manager' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

To upgrade the plugin to a newer version

1. Deactivate the plugin.
2. Delete all the files under '/wp-content/plugins/plugin-manager'.
3. Upload new files to '/wp-content/plugins/plugin-manager'.
4. Activate the plugin.
