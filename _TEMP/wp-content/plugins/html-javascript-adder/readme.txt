== Description ==

This HTML-Javascript Adder allows you to insert HTML, Javascript, simple text and other codes in your sidebar. You can use this plugin to embed flash also. This is very useful if one wants to add some javascripts. You can also add advertisements and arrange it. It is a widget and it can be arranged where ever. No need of editing the sidebar.php file to add javascripts and flash embed codes. Even swfobject javascripts can also be added.

After receiving a volume of email for multiple widget support the latest version of this HTML Javascript Adder comes with a multiple widget support so that you can add more number of javascripts in 9 widgets and rearrange them according to your needs.

= In short : =

You can use this plugin to 

1. Add Google Friend Connect code (The latest trend)
1. Advertisements javascript codes (eg: Google adsense, chitika, adbrite etc)
1. Flash items (eg: Youtube videos, JW Players, Slideshows, flash games made by you etc)
1. Link any files like CSS Stylesheets, Javascript files etc.
1. Social sharing buttons javascript codes (eg: AddThis, shareThis, addtoany, tell-a-friend etc)
1. Write any announcements, some navigation links etc.
1. And more according to your needs.

The another use of this plugin is that, it has a WYSIWYG editor (provided by Simple text editor) which allows you to format the text online.

== Installation ==

1. Download and unzip the current version of the Html-Javascript-adder.
1. Transfer the entire "html-javascript-adder" directory to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it! You're done. 
1. Now goto the widgets settings section and add the widget.
1. If you want to change the number of widgets goto Settings --> Add HTML Javascript and change the number of widgets to be displayed and again go to widgets page and add the javascript in the required widget.

= Upgradation =

1. If you are already using an older version of this plugin, then just deactivate the old plugin and delete the "html-javascript-adder" folder from the plugins directory.
1. Upload the new "html-javascript-adder" to the plugin directory and follow the last two installation steps.

It is highly recommended that you don't overwrite the existing "html-javascript-adder" directory. This new version contains new variables and other new items introduced so it is always better to start as a new one.

== Frequently Asked Questions ==

= What content should be entered in the Text/Javascript/HTML box =

You can enter any script like simple texts, javascripts and HTML Codes. See the description tab for some suggestions.

= Can i enter javascripts/HTML in the "Title" box ? =

No please don't enter. Though, the widget accepts the code the output will be bad and it is not yet tested.

= What to do if i fall into trouble ? =

You will not fall into trouble, it is rare. If you got any trouble, just deactivate the plugin and activate it again.
