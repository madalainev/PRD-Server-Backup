Disables all RSS feeds.

== Description ==
Allows a blog administrator to disable all RSS feeds. (Useful for CMS installations or other situations where RSS is not helpful)

= Version History =
1.0
* Original release.

Background:

Smashing Magazine suggested this hack. (http://www.smashingmagazine.com/2008/12/02/10-useful-rss-hacks-for-wordpress/)

== Installation ==

1. Upload the directory "disable-rss" to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
